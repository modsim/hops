Detailed Description

template<typename Model>
class hops::DynMultimodalModel< Model >

Dynamic multimodal model, which forms weighted linear combinations of arbirtrary models of the same type Model.

A dynamic multimodal model, which is a weighted linear combination of a set of models of the type Model, i.e.

\[ g(x) = \sum_{i=0}^N w_i f_i(x) \]

These models have to implement the same functions as this class does.

Template Parameters
ModelThe model type of the models which form the multimodal model.

Computes the negative log likelihood of the multimodal model.

Assume \( g = g(x) \) is the density of the multimodal model as defined above. This method computes the negative log likelihood as

\begin{align*} -\log g &= -\log \sum_{i=0}^N w_i f_i \\ &= -\log \sum_{i=0}^N w_i \exp\{ \log f_i \} \\ \end{align*}

using the components computeNegativeLogLikelihood method, which has to be provided by Model.
